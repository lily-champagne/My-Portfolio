Infinity Hair Salon

This was a client project where I was asked to design a website for a new hair salon in New York. This fully responsive site was build with the platform, Squarespace and features pages about the staff, a service menu and ability to book appointments. This was created in collaboration with the owner of the salon, Hillary DiBiase.
